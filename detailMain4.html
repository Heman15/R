Data collection Device

I have developed a data collection device using a Feather datalogger board to track the distance from the home position to the destination. A GPS sensor is utilized to collect longitude and latitude data, while an accelerometer is employed to assess the damping effect when the vehicle is on rough terrain.

Skills used: Ardunio, c/c++, sensor fusion, Solidworks.

Data collection Device for autonomous Vehicle

Designed a data collection device to gather data about driving scenarios on Indian roads that can be used to train autonomous vehicles. Pi camera, Rassberry pi board, accelerometer to get speed data, GPS, and steering data using rotary encoder sensor are used. .

Skills used: Python, openCV, c/c++, sensor fusion, Rassberry pi, Solidworks.
